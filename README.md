# Usage

## CommonJS

```js
const add = require('@xiaobo2020/add').default;
// or "const { add } = require('@xiaobo2020/add');"

const result = add(1, 2);

console.log(result);
```

## esModule

```js
import add from '@xiaobo2020/add';
// or "import { add } from '@xiaobo2020/add';"

const result = add(1, 2);

console.log(result);
```
